# Yootnori

A modern implementation of the traditional Korean board game Yut Nori (윷놀이), built natively for visionOS.

## Overview

Yootnori brings the classic Korean folk game to spatial computing with multiplayer support, 3D physics simulation, and immersive experiences on Apple Vision Pro. This project is currently in active development.

## Features

- **SharePlay Integration**: Play together with friends in real-time across Vision Pro devices
- **3D Physics Simulation**: RealityKit-powered dice rolling with realistic physics
- **SwiftUI Interface**: Modern, responsive UI with fluid animations optimized for spatial computing
- **Multiplayer State Management**: Synchronized game state across all players

## Requirements

- visionOS 1.0+
- Xcode 15.0+
- Swift 6.0+

## Status

This project is under active development and not yet available on the App Store.
